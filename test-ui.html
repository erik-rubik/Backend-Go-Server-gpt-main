📋 How to Play:

  1. Connect: Enter a username and click Connect
  2. Wait for Round: Watch the timer for when a new round starts
  3. Submit Message: Send one message per round during the active period
  4. Watch for Winner: A random winner is selected from all submissions
  5. Have Fun: Each round lasts 15 seconds with automatic cycling!
